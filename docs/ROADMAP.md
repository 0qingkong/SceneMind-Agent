# SceneMind Agent 比赛开发路线

提交截止：2026-08-08

## 比赛版核心闭环

```text
拍照 / 上传图片
→ 识别场景中的多个物体
→ 计算基础空间关系
→ 生成结构化场景记录
→ 保存为场景记忆
→ 查询某件物品最后出现的位置
→ 返回图片证据、时间和位置说明
```

## 阶段 1：工程骨架与产品闭环

- 前后端项目初始化
- 图片上传、预览、分析结果展示
- API Schema
- 错误处理
- GitHub 仓库和提交规范

## 阶段 2：真实视觉能力

- 接入目标检测模型
- 返回类别、置信度和边界框
- 前端叠加边界框
- 基于 bbox 计算左右、上下、包含、邻近关系

## 阶段 3：场景记忆与 Agent

- SQLite / PostgreSQL 保存场景
- 保存 ObjectObservation
- 查询物体历史
- last-seen 工具
- 简单 Agent 路由与调用轨迹

## 阶段 4：比赛包装

- 移动端视觉优化
- 测试集和量化指标
- README、架构图、演示视频
- 部署与备份
- 提交材料检查

## 严格范围控制

比赛前不做：

- AI 眼镜硬件接入
- 连续视频流处理
- 多用户复杂权限
- 复杂社交功能
- 自训练大模型
- 大规模分布式系统
